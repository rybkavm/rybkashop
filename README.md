RybkaShop - Modern E-commerce Platform

Современный интернет-магазин с дизайном в стиле Glassmorphism, вдохновленном последними версиями iOS. Проект представляет собой полнофункциональную e-commerce платформу с адаптивным дизайном и интуитивно понятным пользовательским интерфейсом.

Ссылка на работающий проект: https://rybkashop.netlify.app

Функционал магазина:
 - Каталог товаров с 10+ категориями (шапки, бананы, игры для ПК, стулья, блокноты и другие)
 - Карточки товаров с детальной информацией
 - Система скидок (20% товаров со специальными ценами)
 - Промо-слайдер с акциями и новинками
 - Похожие товары и рекомендательная система
 - Фильтрация по категориям
 - Рабочий поиск по товарам

Пользовательский функционал:
 - Регистрация и авторизация
 - Личный кабинет с настройками профиля
 - Избранное (лайки на товарах)
 - Корзина покупок с возможностью изменения количества
 - Промокоды (рабочий промокод "РЫБКА" для 10% скидки на первый заказ)
 - История просмотров

Особенности дизайна:
 - Glassmorphism эффект в стиле iOS
 - Полная адаптивность под все устройства
 - Интуитивная навигация с фиксированными header/footer
 - Плавные анимации и переходы
 - Оптимизированная UX/UI для мобильных устройств

Технологии (стек)
Frontend Core:
 - Vite - быстрый сборщик проекта
 - React 18+ с TypeScript - основной фреймворк
 - React Router DOM - клиентская маршрутизация

Стилизация & UI:
 - Tailwind CSS с настройками Glassmorphism
 - shadcn/ui - компоненты интерфейса
 - Lucide React / React Icons - иконки

State Management & Утилиты:
 - Zustand (useStore.ts) - управление состоянием приложения
 - React Hook Form - работа с формами
 - Кастомные хуки (useDebounce, useToast, use-mobile)

Тестирование & Инструменты:
 - Vitest - тестирование (конфигурация в viteset.config.ts)
 - ESLint - линтинг кода
 - TypeScript - статическая типизация
 - PostCSS - обработка CSS

Сайт полностью адаптирован для:
Мобильные телефоны (320px - 767px)
 - Кастомный бургер-меню с поиском
 - Оптимизированные карточки товаров
 - Скрытие дублирующихся скроллбаров
 - Компактный футер

Планшеты (768px - 1023px)
 - Адаптивные grid-сетки
 - Гибридная навигация
 - Оптимальные размеры интерфейса

Настольные компьютеры (1024px+)
 - Полноценная функциональность
 - Выпадающие меню категорий
 - Многоколоночные layout
 - Расширенные возможности интерфейса

Кроссбраузерность:
 - Современные версии Chrome, Firefox, Safari, Edge
 - Поддержка мобильных браузеров
